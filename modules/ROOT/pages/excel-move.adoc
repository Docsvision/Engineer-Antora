= Переместить шаблон в другую папку

Файл шаблона можно сохранить в любую другую папку. Данная операция возможна только в режиме редактирования шаблона.

.Чтобы переместить файл шаблона:
include::excel-access.adoc[tags=templ]
. Нажмите кнопку *Изменить*, чтобы изменить шаблон.
+
Будет открыто окно _Настройки шаблона_.
+
.Окно "Настройки шаблона"
image::view-template.png[Окно "Настройки шаблона"]
+
. Нажмите кнопку image:buttons/Save.png[Сохранение].
+
. В появившемся окне выберите папку, в которую необходимо сохранить файл шаблона.
. Нажмите кнопку *OK*.
. Закройте окно _Шаблоны_, нажав кнопку *Закрыть*.
