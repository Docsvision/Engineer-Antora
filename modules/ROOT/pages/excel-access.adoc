= Назначить права доступа к шаблону

Права на чтение -- минимальное условие для использования шаблона. Пользователь, которому права на использование шаблона не назначены, не сможет экспортировать представление в Excel.

.Чтобы назначить пользователям права доступа к уже существующему шаблону:
// tag::templ[]
include::partial$excerpts.adoc[tags=menu]
чтобы отобразить список существующих шаблонов Microsoft Excel.
+
.Окно "Шаблоны"
image::template-list.png[Окно "Шаблоны"]
+
. Выберите в списке нужный шаблон.
// end::templ[]
. Откройте стандартное окно назначения прав доступа нажатием кнопки *Доступ* для выбранного шаблона.
. Настройте права доступа требуемым образом.
. Закройте окно _Шаблоны_, нажав кнопку *Закрыть*.
